1. download & install git (one time)
2. set up user name & email_id in git (git cmd ya git bash se)
    git config --global user.name "Archana"
    git config --global user.name "iamsdArchana@gmail.com"

to check user name & email_id connected in git
    git config --global user.name
    git config --global user.email
